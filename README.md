# Вечер покраса у Карповых

Локальный сервис для бронирования мест на вечера покраса. Данные хранятся в Postgres, сайт и API запускаются через Node.js.

## Возможности

- Авторизация через отдельное окно входа/регистрации.
- Первый зарегистрированный пользователь автоматически становится админом.
- Админ может выдавать админ-права другим пользователям.
- Создавать и удалять вечера покраса может только админ.
- Бронирование покрасомест и игровых столов.
- Игровые столы поддерживают двух игроков.
- Отметка оплаты по каждому участнику.

## Запуск

Нужны Docker Desktop и Node.js.

```powershell
cd "C:\Users\Journeymagne\Documents\Трекер посещения покрасавечеров"
powershell -ExecutionPolicy Bypass -File .\start.ps1
```

После запуска сайт будет доступен по адресу:

```text
http://localhost:3000
```

Postgres поднимается через Docker Compose на локальном порту `55432`.

## Файлы

- `server.js` - Node.js сервер, API и инициализация таблиц Postgres.
- `app.js` - клиентская логика сайта.
- `index.html` - разметка страницы и диалогов.
- `styles.css` - темная тема интерфейса.
- `docker-compose.yml` - Postgres для локального запуска.
- `start.ps1` - запуск базы и сайта.

## База данных

Сервис создает таблицы автоматически при старте:

- `users`
- `user_sessions`
- `paint_days`
- `booking_slots`

Если первый пользователь уже был создан до появления админ-ролей, сервер при старте назначит админом самого раннего пользователя, если админов еще нет.
